O projektu


Cilj ovog projekta jeste razviti aplikaciju koja će pomoći u vođenju evidencije o ženskoj košarci u Srbiji. Kao posebne klase se javljaju Nacionalnost, Liga, Tim i u okviru Tima je ugnježdena tabela Igrac

O nama

Proces razvijanja aplikacije


Backend

Projekat je rađen u Javi, u programskom okruženju Spring Tool Suite. Baza podataka koja je korišćena za ovaj projekat napravljena je u okruženju PostgreSQL, a testiranje veb servisa je vršeno preko SoapUI.


Frontend

Sam izgled aplikacije je rađen u okruženju Visual Studio Code. Framework koji je korišćen jeste Angular, besplatni okvir za veb aplikacije sa otvorenim kodom zasnovan na TypeScript-u.
